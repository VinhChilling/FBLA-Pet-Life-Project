# About

Our FBLA Virtual Pet Project, Pet Life, is a game that simulates the effort of caring for a pet while balancing your own life. The goal of the game is to survive until day 30 while maximizing their stats and total 
points accumulated during the playthrough. To achieve this, they must balance and complete all required activities for the pet and their own life in the game. We created this game because we wanted to help young learners
understand the importance of balancing activites and responsibilites in a fun and engaging way such as through caring for a virtual pet.

<!-- EDIT THIS SECTION: List the core functionalities of your application. Use specific, quantifiable points. -->

Interactive Status Bars: Monitor real-time pet metrics (Hunger, Happiness, Energy) that dynamically change over time.

Action System: Users can interact with the pet by Feeding, Playing, and Resting, with corresponding visual and statistical feedback.

Daily & Weekly Requirements: The player must care for the pet while doing activites that benefit their own stats. More info is listed in help page.
